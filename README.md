# WildCard - Poker Trip Planner

A PWA for planning poker tournament trips using Frontier GoWild pass flights.

## Features
- Tournament listings from multiple sources (Poker Atlas, Hendon Mob, Card Player)
- GoWild flight pricing from ORD/MDW
- Hotel cost estimates
- Total trip cost calculator
- Search, sort, and filter
- Save trips for comparison
- AI trip planning assistant

## Setup
1. Clone the repo
2. `npm install`
3. `npm start` for development
4. `npm run build` for production

## Deployment
Auto-deploys to Vercel on push to main branch.
